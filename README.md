# Simulador de Oscilador Armónico Amortiguado

Visualización de las diferentes **soluciones analíticas** del comportamiento de un sistema masa-resorte con amortiguamiento en diferentes regímenes físicos. 
Posibles mejoras: mejor interacción y estética de la visualización.

## Funcionalidades
- **Selección Automática de Régimen:** El código identifica si el sistema es subamortiguado, sobreamortiguado o tiene amortiguamiento crítico según los parámetros ingresados ($b^2 - 4mk$).
- **Visualización Dinámica:** Grafica la posición en función del tiempo y permite observar la envolvente de la amplitud, a su vez se muestra en loop el comportamiento de la masa.
